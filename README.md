# ClickGUI Mod

A Fabric mod for Minecraft 1.21.4 written in Kotlin. Features a World Tweaks ClickGUI with time control and snow effect.

## Features

- 🎨 Beautiful ClickGUI with rounded corners and smooth animations
- ❄️ Animated snow particles in background
- 🌍 World Tweaks panel with time control
- ⌨️ Keybinding support (Right Shift to toggle)
- 🎯 Toggle controls for World Tweaks and Time Control
- 🕐 Custom time slider with visual feedback

## Building

```bash
./gradlew clean build
```

The compiled JAR will be in `build/libs/untitled-1.0.0.jar`

## Installation

1. Download the compiled JAR from `build/libs/`
2. Place it in your Minecraft `mods/` folder
3. Make sure you have installed:
   - Fabric Loader 0.19.2+
   - Fabric API 0.119.4+
   - Fabric Language Kotlin 1.13.12+
4. Launch Minecraft with Fabric

## Controls

- **Right Shift** - Toggle ClickGUI menu
- **Mouse** - Interact with UI elements (toggles and sliders)
- **Escape** - Close menu

## Requirements

- Java 21 or later
- Minecraft 1.21.4
- Fabric Loader 0.19.2+
- Fabric Language Kotlin 1.13.12+
- Fabric API 0.119.4+
